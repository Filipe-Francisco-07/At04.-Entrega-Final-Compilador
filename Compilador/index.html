Compilador


O compilador fará as análises Léxica, Sintática e Semântica da sua entrada!

Tokens aceitos:


id (A-Z ou A-Z 0-9 sem ser palavras reservadas), if (if), print (print)

for (for), constantes (0-9), mais(+), incremento(++), menos(-)

decremento(--), barra(/), pv(;), maior(>), maiorigual(>=), menor(<)

menorigual(<=), igualdade(==), ap((), fp()), ac({), fc(})

programa(programa), v(,), char(char), int(int), igual(=)

programa nomeprog ( int a ) { if ( a > 10 ) { int x ; } }
